Metropole-backend

## Installation

Server start

```bash
npm run dev
```

In order to start PostgreSQL database in window run:

Start

```bash
pg_ctl.exe start -D "C:\Program Files\PostgreSQL\12\data"
```

Restart

```bash
pg_ctl.exe restart -D "C:\Program Files\PostgreSQL\12\data"
```

NGROK

```bash
ngrok http 8080
```
